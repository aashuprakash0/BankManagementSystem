<h1 align="center">🏦 Bank Management System</h1>

<p align="center">A Desktop Banking Application built using Java, Swing, JDBC, and MySQL.</p>

---

## 🚀 Features
- Account creation & customer registration  
- Deposit & withdrawal  
- Balance inquiry  
- Transaction history  
- User login & authentication  
- Input validation and error handling  
- Swing-based GUI  
- JDBC-backed MySQL database

## 📂 Project Structure
```
/src
 ├── Main.java
 ├── Login.java
 ├── Dashboard.java
 ├── AccountOperations.java
 ├── DatabaseConnection.java
 └── TransactionHistory.java

/database
 └── bank.sql
```

---

## 🔧 How to Run
1. Clone the repository  
2. Open the project in Eclipse or IntelliJ  
3. Import the `bank.sql` file into MySQL  
4. Update DB username/password in `DatabaseConnection.java`  
5. Run the program
